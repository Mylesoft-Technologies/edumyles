# EduMyles — Landing Page

Marketing and public-facing website hosted at `edumyles.com`.

**Separate** from the school app (`{slug}.edumyles.com`).

## Tech
- Next.js App Router
- Tailwind CSS
- shadcn/ui

## Routes
- `/` — Homepage
- `/pricing` — Pricing tiers
- `/features` — Feature overview
- `/contact` — Contact / demo request
- `/blog` — Blog (optional Phase 2)
